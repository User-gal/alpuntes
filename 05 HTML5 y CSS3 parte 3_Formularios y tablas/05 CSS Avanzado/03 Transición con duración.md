¿Cómo configurar la transición visual de un elemento que dura 3 segundos?

- Alternativa correta
    
    ```makefile
    change: 3s
    ```
    
- Alternativa correta
    
    ```css
    transition: 3s;
    ```
    
    ¡Alternativa correcta! Utilizamos la propiedad `transition` para proporcionar una transición visual de un elemento.
