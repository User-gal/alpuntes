Dada la siguiente lista:

- Ítem amarillo
- Ítem verde
- Ítem rojo

¿Cuál sería la estructura HTML para crearla?

- Alternativa correta
    
    ```css
    <ol>
    <li>Ítem amarillo</li>
    <li>Ítem verde</li>
    <li>Ítem rojo</li>
    </ol>
    ```
    
- Alternativa correta
    
    ```css
    <ol>
    <li>Ítem amarillo
    Ítem verde
    Ítem rojo</li>
    </ol>
    ```
    
- Alternativa correta
    
    ```css
    <ul>
    <li>Ítem amarillo</li>
    <li>Ítem verde</li>
    <li>Ítem rojo</li>
    </ul>
    ```
    
    ¡Alternativa correcta! La etiqueta para usar listas no ordenadas es `<ul>`, y cada ítem de la lista necesita ser marcado por un `<li>`
