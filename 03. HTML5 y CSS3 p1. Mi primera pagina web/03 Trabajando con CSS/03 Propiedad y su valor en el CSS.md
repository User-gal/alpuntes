¿Cuál es la sintaxis correcta para separar la propiedad y su valor en el CSS?

- Alternativa correta
    
    `El carácter "=" (text-align= center)`
    
- Alternativa correta
    
    `El carácter "´;" (text-align; center)`
    
- Alternativa correta
    
    `El carácter ":" (text-align: center)`
    
    ¡Alternativa correcta! El carácter `:` sirve para atribuir un valor a una propiedad.
